# 🌾 SmartFlow — AgriSCM System
> **Dự án:** Quản trị dự án xây dựng Hệ thống Thông tin Quản lý Chuỗi cung ứng Nông sản.
> **Học phần:** Quản trị Dự án Hệ thống Thông tin (IT Project Management - ITS344).

## 📖 Bối cảnh dự án (Business Context)
Hệ thống web-based hỗ trợ quản lý và truy xuất nguồn gốc chuỗi cung ứng lúa gạo đặc sản (ST24, Đài Thơm 8) tại khu vực Buôn Triết, Lắk, Đắk Lắk. Hệ thống giải quyết các bài toán về:
- Truy xuất nguồn gốc bằng mã QR.
- Quản lý kho bãi và thất thoát.
- Quản lý đơn hàng và minh bạch giá cả giữa nông hộ, hợp tác xã và thương lái.

## 🛠 Tech Stack (Công nghệ sử dụng)
- **Framework:** PHP (Laravel 10.x)
- **Database:** MySQL 8.0
- **Frontend:** HTML, CSS, JavaScript (Bootstrap 5)
- **Môi trường Dev:** MAMP (Localhost)
- **Quản lý dự án:** Jira, Confluence, GitHub

## 👥 Đội ngũ Phát triển (Project Team)

## 📁 Cấu trúc Thư mục (Directory Structure)
- `/docs`: Tài liệu dự án phân chia theo 7 chương của PMBOK.
- `/assets`: Hình ảnh, biểu đồ Mermaid/Draw.io, mockups.
- `/meeting_minutes`: Lưu trữ các biên bản họp nhóm định kỳ.
- `/src`: Mã nguồn chính của hệ thống (Laravel).
